# ParserLalafo
Парсер Lalafo (Kyrgyzstan) с сохранением в Excel и менеджером параллельного запуска
## Описание

Этот проект позволяет:

- Парсить категории и подкатегории с сайта [Lalafo.kg](https://lalafo.kg)
- Собрать ссылки на объявления
- Извлечь данные пользователя (телефон, имя, город)
- Сохранять результаты в Excel
- Объединять Excel-файлы и делить их на части по 20 000 строк
- Параллельно запускать несколько скриптов с разными URL

## Установка

1. Клонируйте репозиторий:

git clone https://github.com/yourusername/yourproject.git
cd yourproject

2. Создайте виртуальное окружение (опционально):

python -m venv venv
source venv/bin/activate      # Linux / Mac

venv\Scripts\activate         # Windows

3. Установите зависимости:

pip install -r requirements.txt


Примечания
Скрипты поддерживают Unicode (UTF-8), чтобы корректно выводились эмодзи и кириллица.
Максимальное количество параллельных процессов в менеджере можно изменить через переменную MAX_PROCESSES.
